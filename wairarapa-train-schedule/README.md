# Wairarapa Train Schedule

A real-time train schedule web application for the Wairarapa line in Wellington, New Zealand. Shows live departure times, delays, and service notices for Wellington, Petone, and Featherston stations.

## Setup

### 1. Install Dependencies

```bash
npm install
```

### 2. Configure API Key

1. Sign up for a free API key at [Metlink Open Data](https://opendata.metlink.org.nz/)
2. Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```
3. Add your API key to `.env`:
   ```
   REACT_APP_METLINK_API_KEY=your_metlink_api_key_here
   ```

### 3. Start Development Server

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Security

- API keys are stored in environment variables (not in source code)
- The `.env` file is excluded from version control
- Use `.env.example` as a template for new environments
